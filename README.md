# SampleiOSMap

iOSMap の実験用リポジトリ

## Description

- 自分の位置の地図を表示させる
- 画面をロングタップするとピンを留めることができる
- ピン留めした箇所は任意の画像(牛)で表示する
- そのピンを押すと、タイトル[LongPressPin],サブタイトル[sub],右側に OK ボタンを表示させる
- OK ボタンを押すと「OK ボタン押下後 OK ボタン押下後のポップアップ」と記載されたポップアップを表示させる
